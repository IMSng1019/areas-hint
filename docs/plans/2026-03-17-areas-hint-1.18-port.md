# areas-hint 1.18 Port Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Adapt `areas-hint` from its current Fabric baseline to Minecraft 1.18 while preserving behavior and making only compatibility changes.

**Architecture:** The port will be done in thin layers. First, lock the dependency baseline and metadata constraints. Then use repeated build runs to reveal incompatibilities, capturing them in a running to-do list and fixing one error at a time. If the error surface is large, constrain edits by module so each compatibility migration stays local and verifiable.

**Tech Stack:** Java, Gradle, Fabric Loom, Fabric Loader, Fabric API, Yarn mappings, Sponge Mixin, Context7 docs.
